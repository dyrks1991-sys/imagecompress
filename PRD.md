---

## Vision

Eliminate the friction of image optimization for every creator — zero uploads, zero accounts, instant results.

## Description

A fully client-side image compression and conversion tool. Users drag-and-drop images, choose output format (JPEG / WebP / PNG) and quality (1–100), see a live before/after preview with file-size savings, then download the result. All processing happens in the browser via the Canvas API — nothing touches a server.

## MVP Features

| ID | Feature | Priority | Acceptance Criteria |
|---|---|---|---|
| F1 | **Image Upload** | P0 | Accepts JPG, PNG, WebP, GIF via drag-drop or file picker. Shows file name and original size. |
| F2 | **Format Selector** | P0 | Radio buttons: JPEG | WebP | PNG. Default JPEG. |
| F3 | **Quality Slider** | P0 | Range 10–100. Default 80. Updates size estimate within 200ms. |
| F4 | **Live Preview** | P0 | Side-by-side: original (left) vs. compressed (right). Both show dimensions and file size. |
| F5 | **Download** | P0 | Triggers file download with name pattern: imagecompress-YYYYMMDD-HHMMSS.ext |
| F6 | **Savings Display** | P0 | Shows: original KB → new KB (X% smaller). Red if compression increased file size. |

## User Stories

### US1 — blogger

**As a** blogger, **I want to** I want to compress a JPEG before uploading to WordPress so my page loads faster.

**Done when:** Upload JPG → choose JPEG output at 80% → download in < 3s.

### US2 — developer

**As a** developer, **I want to** I want to convert a PNG to WebP so I can use next-gen formats in my web app.

**Done when:** Upload PNG → select WebP → download.

### US3 — designer

**As a** designer, **I want to** I want to see exactly how much quality I lose at different compression levels.

**Done when:** Side-by-side preview updates within 300ms of slider change.

### US4 — privacy-conscious user

**As a** privacy-conscious user, **I want to** I want to compress images without uploading them to a server.

**Done when:** No network request is made during compression — verified in DevTools.

### US5 — content creator

**As a** content creator, **I want to** I want the compressed filename to include today's date so I can organize my files.

**Done when:** Downloaded file is named imagecompress-YYYYMMDD-HHMMSS.jpg.
